# Counter Project

A simple, interactive counter web application built to practice and solidify core JavaScript basics, such as DOM manipulation and event handling.

## 🚀 Features
* Increment and decrement the counter value.
* Reset the counter back to zero.
* *[Optional]* Dynamic color changes (e.g., text turns red for negative numbers, green for positive).

## 🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript
* DOm

## 💡 What I Learned
* How to select and manipulate HTML elements using JavaScript.
* Working with event listeners (`addEventListener`).
* Managing application state (keeping track of the count).
